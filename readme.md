おんJ艦これ部スレッドテンプレート作成ソフト 作成:17/02/26/
作成者:朝霧 粉山

このソースはパブリックドメインです。
自由に改変、再配布して構いません。
なんかあったらtwitter:@Temporal_Towerまで。

以下使い方
1:gccとかでコンパイルする
2:コマンドライン上で実行する
3:Start?と聞かれるので作り始めたいスレ番号を入力する
4:End?と聞かれるので作り終わりたいスレ番号を入力する
5:Comment out Start?と聞かれるのでコメントアウトを始めたいスレ番号を入力する
6:実行ファイルが置かれているところにテキストファイルが作成されるので中身をwikiに貼り付ける
7:おーわりー！かんたんだね！
